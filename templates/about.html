About SweGen ExAC

The SweGen Variant Frequency Browser uses the browser software developed for the ExAC Browser developed by the Exome Aggregation Consortium (ExAC), Cambridge, MA [August, 2016], with modifications. The original ExAC Browser project is available on GitHub.

For questions, comments, etc., regarding the SweGen cohort dataset, please contact swegen@scilifelab.se.

For inquiries about this website, please use swefreq@scilifelab.se.

For technical bug reports relating to the browser software, please file an issue on our GitHub issue page.
